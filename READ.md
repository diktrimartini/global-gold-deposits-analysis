# Global Gold Deposits Analysis

## Project Overview
This project analyzed the global distribution of major gold deposits using the USGS Major Mineral Deposits dataset.

## Dataset
Source: https://mrdata.usgs.gov/major-deposits/

## Tools used
- Python
- pandas
- matplotlib
- dbfread

## Analysis Performed
- Commodity analysis
- Gold deposit filtering
- Country distribution analysis
- Deposit type analysis

## Key Findings
- South Africa contains the highest number of gold deposits in the dataset.
- Hydrothermal deposits dominate global gold mineralization.
- Surficial and igneous systems are also significant.

## Output
- pure_gold_deposits.csv
- Top gold country visualization
- Gold deposits type visualization
